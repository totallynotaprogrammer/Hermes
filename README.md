# The Watchtower — Signal Intelligence

A personal signal intelligence dashboard for tracking structural shifts across domains sitting at the edge of what is currently changing fastest in the world.

Built as a self-contained HTML file. No server. No login. No dependencies beyond a browser.

---

## What's in this repository

**`watchtower-v2.2.html`** — The main dashboard. Tracks five signal channels plus a Fortean anomaly monitor running on its own epistemological protocol.

**`watchtower-companion.html`** — Field Notes. Explains the architecture, the philosophy behind each channel, and how to read and adapt the system.

---

## The Five Channels

- **AI Governance + Biosecurity** — Tracking the shift from policy principles into embedded technical constraints
- **Emerging Tech Convergence** — Watching the tight coupling of AI, biotech, and automation stacks
- **Psychedelic Research** — Tracking the structural split between regulated therapeutics and engineered neuroactive compounds
- **Neuro + Mental Health Tech** — Three parallel treatment modality tracks converging toward a single inflection point

And a sixth channel running on different rules entirely:

- **Fortean Anomaly Monitor** — Tracks recurrence and pattern, not truth claims. Named after Charles Fort, who believed the act of noticing what gets discarded is itself important data.

---

## How to use it

Download either HTML file and open it in any browser. That's it.

The Companion file is the right place to start if you're new to the system — it explains the logic, the sourcing protocols, and how to adapt the architecture for your own domains.

---

## Philosophy

> *"The architecture is the point, not the content."*

The Watchtower is shared as a lens and a template — not as a set of conclusions. The specific domains tracked reflect one curator's judgment about where structural transitions are underway. Your domains might be different. The system is designed to be rebuilt.
